# Teaching-Management-System
该教学管理系统基于Spring Boot和Vue.js开发，旨在提升教育管理效率。系统支持课程安排、学生信息管理、成绩录入及在线作业发布等功能。教师和学生可通过平台进行互动，查看课程进度和成绩，方便信息共享。此外，系统具有用户权限管理和数据安全保护，确保教学信息安全可靠，助力学校实现数字化管理。
